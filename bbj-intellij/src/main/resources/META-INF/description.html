BBj Language Support brings comprehensive IDE features for BBj development to IntelliJ IDEA.

Features:

  • Syntax highlighting with TextMate grammars
  • Real-time error diagnostics
  • Code completion for BBj keywords and Java classes
  • Go-to-definition and hover documentation
  • Signature help for method calls
  • Java interop intelligence for BASIS.BBjAPI classes

Requirements:

  • Node.js 18+ (for language server)
  • BBj installation (optional, for enhanced features)
